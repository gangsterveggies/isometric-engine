Isometric Engine
================

A SDL based Isometric Tile Engine.

Compilation
===========
How to compile and run:
```
git clone git@github.com:gangsterveggies/isometric_engine.git
cd isometric_engine
git submodule init
git submodule update
cmake .
make
./isoengine
```
